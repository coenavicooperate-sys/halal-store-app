# Halal Store App — 入力マニュアル / Input Manual

店舗情報を入力する方向けのガイドです。  
This guide is for users who enter store information into the form.

---

# 日本語

## 1. はじめに

1. 管理者から受け取った**URL**をブラウザで開く
2. **アクセスコード**を入力して「入力」をクリック
3. フォームが表示されたら入力開始

**言語切り替え：** 画面右上の「English」「日本語」ボタンで切り替えられます。

---

## 2. 入力手順（ステップ順）

### Step 1：基本情報

| 入力欄 | 必須 | 入力例・注意 |
|--------|------|--------------|
| 店舗名（GoogleMap登録名） | ○ | Googleマップに表示されている正式な名前をそのまま入力 |
| お店の電話番号 | ○ | 03-1234-5678 など |
| カテゴリー | ○ | 一覧から選択（ラーメン、日本料理、カフェなど） |
| 担当者名 | - | 登録担当者の名前 |
| メールアドレス | ○ | 連絡可能な有効なメールアドレス |

---

### Step 2：店舗情報

| 入力欄 | 必須 | 入力例・注意 |
|--------|------|--------------|
| 営業時間 | - | 例：月〜日 11:00〜22:00 / 祝日 10:00〜21:00 |
| 定休日 | - | 例：火曜日、なし |
| 最寄り駅 | - | 例：新宿駅 徒歩5分 |

---

### Step 3：設備・対応

| 入力欄 | 必須 | 入力例・注意 |
|--------|------|--------------|
| 対応言語 | - | アラビア語、中国語、英語、インドネシア語、マレー語、ウルドゥー語から選択（複数可） |
| Wi-Fi | - | 利用可能 / 利用不可 のいずれか |
| 決済方法 | ○ | 1つ以上選択（現金、Visa、Mastercard、Alipayなど） |
| ハラル対応レベル | ○ | 完全ハラル認証済み / ムスリムフレンドリー / ハラルメニューあり などから選択 |
| 調理配慮レベル | ○ | 専用キッチン / 専用調理器具 / ハラル専用調理エリア などから選択 |

---

### Step 4：写真アップロード

**TOP写真（3枚必須）**
- 外観・料理・内観など、お店の魅力が伝わる写真を3枚
- 推奨：480×480px の正方形（スマホでも可）
- 形式：JPG、PNG、WebP（各10MB以下）

**認証写真（最大3枚）**
- 「完全ハラル認証済み」を選んだ場合は**1枚以上必須**
- ハラル認証証書の写真など
- 推奨：540×720px の縦型
- 形式：JPG、PNG、WebP（各10MB以下）

---

### Step 5：こだわり（1セット以上必須）

各こだわりには以下を入力してください：

| 項目 | 必須 | 入力例 |
|------|------|--------|
| こだわり写真 | ○ | 写真をアップロード |
| 表題 | ○ | 例：「厳選食材」 |
| こだわり説明 | ○ | 例：「国産の野菜を中心に、新鮮な素材で仕上げています。」 |

最大3セットまで登録できます。写真は縦型（540×720px推奨）がきれいに表示されます。

---

### Step 6：メニュー情報（1品目以上必須）

各メニューには以下を入力してください：

| 項目 | 必須 | 入力例 |
|------|------|--------|
| メニュー写真 | ○ | 料理の写真をアップロード |
| メニュー名 | ○ | 例：「ハラル特製ラーメン」 |
| メニュー説明 | - | 例：「鶏ガラベースのあっさりスープ」 |

最大3品目まで登録できます。写真は縦型（540×720px推奨）がきれいに表示されます。

---

### Step 7：内観・外観写真（1枚以上必須）

- 店内の様子や外観の写真を1〜5枚まで
- 推奨：540×720px の縦型
- 形式：JPG、PNG、WebP（各10MB以下）

---

## 3. 下書きを保存する場合

- 途中で保存したい場合は「下書き」セクションを開く
- 下書き名（例：店舗名）を入力して「下書き保存」をクリック
- **注意：写真は保存されません。読み込み後に再度アップロードしてください。**

続きから入力する場合：
- 下書き一覧から選んで「下書き読み込み」をクリック

---

## 4. 送信の流れ

1. 必須項目をすべて入力し、写真をアップロードする
2. 画面下部の「**確認して送信**」ボタンをクリック
3. エラーがあれば修正し、再度「確認して送信」をクリック
4. 確認画面が表示されたら、内容をよく確認する
5. 「**送信する**」ボタンをクリック
6. **1〜2分待つ**（ページを閉じないでください）
7. 「送信が完了しました！！」と表示されたら完了

**※ 送信後は修正できません。送信前に必ず内容を確認してください。**

---

## 5. 写真の目安（まとめ）

| 種類 | 枚数 | 推奨サイズ |
|------|------|------------|
| TOP写真 | 3枚 | 480×480（正方形） |
| 認証写真 | 0〜3枚 | 540×720（縦型） |
| こだわり | 1〜3セット | 540×720（縦型） |
| メニュー | 1〜3品目 | 540×720（縦型） |
| 内観・外観 | 1〜5枚 | 540×720（縦型） |

すべて JPG / PNG / WebP 形式、1枚あたり最大10MB まで。

---

## 6. よくあるエラーと対処

| 表示 | 対処法 |
|------|--------|
| アクセスコードが正しくありません | 管理者に正しいコードを確認する |
| 店舗名は必須です | 店舗名を入力する |
| 電話番号は必須です | 電話番号を入力する |
| メールアドレスは必須です | メールアドレスを入力する |
| 有効なメールアドレスを入力してください | 正しい形式（例：store@example.com）で入力する |
| 決済方法を1つ以上選択してください | 決済方法を1つ以上選ぶ |
| TOP写真は3枚すべて必要です | 3枚すべてアップロードする |
| こだわりは最低1セット必要です | 写真・表題・説明をすべて入力する |
| メニューは最低1つ必要です | 写真とメニュー名を入力する |
| 内観・外観写真は最低1枚必要です | 1枚以上アップロードする |
| 認証写真が1枚以上必要です | 「完全ハラル認証済み」の場合は認証写真をアップロードする |
| 無効な画像形式 | JPG、PNG、WebP のいずれかでアップロードする |
| ファイルが大きすぎます | 10MB以下の画像をアップロードする |

# English

## 1. Getting Started

1. Open the **URL** provided by the administrator in your browser
2. Enter the **access code** and click **Enter**
3. When the form appears, you can start entering data

**Language:** Use the **English** / **日本語** buttons at the top-right to switch languages.

---

## 2. Input Steps (In Order)

### Step 1: Basic Information

| Field | Required | Example / Notes |
|-------|----------|-----------------|
| Store Name (Google Maps Listing) | Yes | Enter the exact name as shown on Google Maps |
| Store Phone Number | Yes | e.g. 03-1234-5678 |
| Category | Yes | Select from the list (Ramen, Japanese Food, Café, etc.) |
| Contact Person Name | No | Name of the person responsible for registration |
| Email Address | Yes | A valid, working email address |

---

### Step 2: Business Information

| Field | Required | Example / Notes |
|-------|----------|-----------------|
| Business Hours | No | e.g. Mon–Sun 11:00–22:00 / Holidays 10:00–21:00 |
| Regular Holiday | No | e.g. Tuesdays, None |
| Nearest Station | No | e.g. Shinjuku Station, 5 min walk |

---

### Step 3: Facilities & Services

| Field | Required | Example / Notes |
|-------|----------|-----------------|
| Languages Available | No | Select one or more: Arabic, Chinese, English, Indonesian, Malay, Urdu |
| Wi-Fi | No | Available or Not Available |
| Payment Methods | Yes | Select at least one (Cash, Visa, Mastercard, Alipay, etc.) |
| Halal Compliance Level | Yes | Fully Halal Certified, Muslim-Friendly, Halal Menu Available, etc. |
| Preparation Transparency | Yes | Separate Kitchen, Separate Utensils, Dedicated Halal Area, etc. |

---

### Step 4: Photo Upload

**Top Photos (3 required)**
- 3 photos that showcase your store (exterior, food, interior)
- Recommended: 480×480px square (smartphone photos also OK)
- Format: JPG, PNG, WebP (max 10MB each)

**Certification Photos (up to 3)**
- **At least 1 required** if you selected "Fully Halal Certified"
- Photos of Halal certification documents, etc.
- Recommended: 540×720px portrait
- Format: JPG, PNG, WebP (max 10MB each)

---

### Step 5: Highlights (≥1 set required)

For each highlight, enter:

| Item | Required | Example |
|------|----------|---------|
| Highlight Photo | Yes | Upload a photo |
| Title | Yes | e.g. "Premium Ingredients" |
| Description | Yes | e.g. "We use fresh, domestic vegetables and premium ingredients." |

Up to 3 highlights. Portrait photos (540×720px recommended) display best.

---

### Step 6: Menu Information (≥1 item required)

For each menu item, enter:

| Item | Required | Example |
|------|----------|---------|
| Menu Photo | Yes | Upload a photo of the dish |
| Menu Name | Yes | e.g. "Halal Special Ramen" |
| Menu Description | No | e.g. "Light chicken-based soup" |

Up to 3 menu items. Portrait photos (540×720px recommended) display best.

---

### Step 7: Interior / Exterior Photos (≥1 required)

- 1 to 5 photos of the interior or exterior
- Recommended: 540×720px portrait
- Format: JPG, PNG, WebP (max 10MB each)

---

## 3. Saving a Draft

- To save your progress, open the **Draft** section
- Enter a draft name (e.g. store name) and click **Save Draft**
- **Note: Photos are not saved. You must re-upload them after loading.**

To continue later:
- Select a draft from the list and click **Load Draft**

---

## 4. Submission Flow

1. Fill in all required fields and upload photos
2. Click the **Confirm & Submit** button at the bottom
3. If there are errors, fix them and click **Confirm & Submit** again
4. When the confirmation screen appears, review the content carefully
5. Click the **Submit** button
6. **Wait 1–2 minutes** (do not close the page)
7. When "Submission complete!!" appears, you are done

**※ Data cannot be modified after submission. Please verify everything before submitting.**

---

## 5. Photo Requirements (Summary)

| Type | Count | Recommended Size |
|------|-------|-------------------|
| Top Photos | 3 | 480×480 (square) |
| Certification | 0–3 | 540×720 (portrait) |
| Highlights | 1–3 sets | 540×720 (portrait) |
| Menu | 1–3 items | 540×720 (portrait) |
| Interior/Exterior | 1–5 | 540×720 (portrait) |

All formats: JPG / PNG / WebP, max 10MB per image.

---

## 6. Common Errors and Solutions

| Message | Solution |
|---------|----------|
| Incorrect access code | Verify the correct code with the administrator |
| Store Name is required | Enter the store name |
| Phone Number is required | Enter the phone number |
| Email Address is required | Enter the email address |
| Please enter a valid email address | Enter a valid format (e.g. store@example.com) |
| At least one payment method is required | Select at least one payment method |
| All 3 Top Photos are required | Upload all 3 photos |
| At least 1 Highlight (photo, title, description) is required | Enter photo, title, and description for at least one highlight |
| At least 1 menu (photo, name) is required | Enter photo and name for at least one menu |
| At least 1 interior/exterior photo is required | Upload at least 1 photo |
| At least 1 certification photo is required | Upload certification photo if "Fully Halal Certified" |
| Invalid image format | Upload JPG, PNG, or WebP only |
| File too large | Upload images under 10MB each |
